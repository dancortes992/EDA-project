# EDA Practicum Project
### Background

This project is part of the data science Tripleten bootcamp practicum concerning EDA.

## Objective

To analyze the provided datasets and gather insights on costumers shopping habits and order placment such as:
- Number of orders placed per day by costumer.
- Frequency of purchases.
- Distributions of reorders, weekly purchases, etc..
- Top weekly purchases.
- Top products.

## Software

**Libraries:** _Python_, _matplotlib_
